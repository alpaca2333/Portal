# Quant Research Orchestration

This directory tracks the ongoing quant strategy research loop.

## Files
- `state.json`: current round / best metrics / manager status
- `manager_notes.md`: manager decisions and improvement directions
- `round_*_summary.md`: per-round manager review summaries

## Goals
- Annual return target: >= 15%
- Sharpe target: >= 1.0
- Max drawdown target: better than -25%

The manager reviews reports, requests new rounds, and escalates code errors to the programmer.
