# lgbm_ensemble_adaptive_v3 回测报告

## 1. 策略概述

### 策略思路

自适应集成选股 V3：在 V2 基础上新增小盘暴露上限和前瞻性风控信号。核心选股层沿用 V2（双模型 44 因子 + 共识过滤）。

### V3 核心改进

1. **小盘暴露上限**：小盘股(市值后30%)总权重上限30%(压力时15%)；当 vol_scale < 0.50 时自动压制 small_cap_bonus
2. **前瞻性风控信号**：
   - 市场宽度加速度（二阶导数 < -0.08）
   - 跌停比率（> 3% 触发预警）
   - 成交量萎缩（< 60% 均量触发预警）
   - 综合风控评分直接调节仓位，下限 40%
3. **风控驱动选股过滤**：当风控乘数 < 80% 时，直接从选股池剔除市值后25%的小盘股

### 继承自 V2

- 波动率目标制（目标 18%）
- 三级趋势确认（短/中/长期）
- 自适应止损（-20% ~ -8%）
- 风格自适应 + 训练优化 + 组合优化
