# lgbm_cross_sectional_v7 回测报告

## 1. 策略概述

### 策略思路

基于 LightGBM 的截面选股策略（V7），在 V4 基础上增加基本面再排序，温和提升价值/质量因子的影响力，同时保留 V4 的核心 ML 选股能力。

### 核心设计

1. **截面排序标签**：预测相对排名（百分位），消除收益分布的非平稳性
2. **3 年滚动训练窗口**：训练集按标签结束日 purge，避免前视偏差
3. **行业约束选股**：ML 分数前 5%，每行业最多 3 只
4. **持仓缓冲带**：在持股票 ML 分数加 0.5σ，降低换手
5. **选股范围**：沪深主板，自由流通市值前 85%
6. **V7 基本面再排序**：ML 候选池内按 70% ML排名 + 30% 基本面排名 综合排序（基本面 = inv_pb + roe_ttm + roa_ttm + dv_ttm + turnover_20 的 rank 均值）
