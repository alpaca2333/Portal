# lgbm_index_enhancement 回测报告

## 1. 策略概述

### 策略思路

基于 LightGBM 的指数增强策略（V2），以 000300.SH 为基准，在基准权重的基础上，根据 ML + 基本面综合打分进行主动偏离（超配/低配），同时保持行业中性以控制跟踪误差。

### V2 改进

1. **超额收益标签**：训练标签从绝对收益改为相对基准的超额收益
2. **行业中性约束**：每个行业偏离不超过 ±3%
3. **优化权重分配**：基于 z-score 的主动偏离，替代简单 Top-N 选股
4. **成分股内训练**：只用沪深300成分股样本训练模型
5. **换手率约束**：单期单边换手率上限 30%

### 核心设计

1. **选股池与基准**：严格限制在 000300.SH 成分股内，初始权重为基准权重
2. **主动偏离**：单只股票主动偏离不超过 1.5%
3. **行业中性**：在每个申万一级行业内，超配和低配的权重总和趋近 0
4. **综合打分**：70% ML排名 + 30% 基本面排名
