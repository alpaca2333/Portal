# lgb_v2_industry_neutral 回测报告

**策略描述**: LightGBM v2: 行业均衡选股 + Expanding window + 回归/排序双头集成 + 样本衰减加权。  

## 策略背景与改动原因

### 动机

V1 策略的核心问题：
1. 全局 top 5% 选股导致行业集中度过高，在行业轮动中回撤严重。
2. 固定滚动窗口丢弃大量历史数据，样本量不足以支撑 tree model。
3. 单一回归模型只优化 MSE，无法保证排序质量。

### V2 核心设计

1. **行业均衡选股**: 每个行业内取 ML 得分前 N 名，跨行业等比例选取。消除行业 beta 暴露。
2. **Expanding window + 衰减加权**: 训练集随时间扩展，但对远期样本施加指数衰减权重（半衰期 ~1.5 年），兼顾样本量与时效性。
3. **双头集成**: 回归头优化 Huber loss（对极端值鲁棒），排序头优化 LambdaRank（直接优化 NDCG 排序质量），最终 rank-space 混合。
4. **行业中性化标签**: 训练目标是行业内超额收益，而非绝对收益。
5. **Purge + Embargo**: 训练窗口与测试窗口间设 1 个月隔离带。
6. **Z-score 特征归一化**: 保留距离信息（v1 的 rank 只保留序关系）。

## V2 模型架构

- **训练方式**: Expanding window（训练集随时间扩展，不丢弃历史数据）
- **初始训练**: 4 年，每 6 个月扩展一次
- **Embargo**: 1 个月隔离带
- **样本权重**: 指数衰减，半衰期 540 天
- **特征数**: 15 个，截面 z-score 归一化
- **训练步数**: 17

## V1 vs V2 对比

| 维度 | V1 | V2 |
|------|-----|-----|
| 训练窗口 | 3年固定滚动 | Expanding + 衰减加权 |
| 标签 | 截面 rank 归一化 | 行业中性化超额收益 |
| 特征归一化 | 截面 percentile rank | 截面 z-score (±3σ clip) |
| 模型 | 单一 LGBMRegressor (MSE) | 回归(Huber) + 排序(LambdaRank) 集成 |
| 选股 | 全局 top 5% + 行业上限5只 | 行业内排名 → 等比例跨行业选取 |
| 样本权重 | 无 | 指数衰减（半衰期 540 天） |
| Purge | label_end_date 检查 | label_end_date + 1个月 embargo |
| 评估指标 | Rank IC only | Rank IC + 分位价差 + 单调性 |
