# lgbm_cross_sectional_v5 回测报告

## 1. 策略概述

### 策略思路

基于 LightGBM 的截面选股策略（V5），在 V4 基础上重点优化特征工程，去除冗余因子、引入高信息量新因子。

### V5 vs V4 特征工程改进

#### 移除的冗余特征（5 个）

| 移除特征 | 原因 |
|---------|------|
| `mom_3_1` | 与 `mom_12_1` 高度相关（r>0.7），信息冗余 |
| `mom_6_1` | 与 `mom_12_1` 高度相关（r>0.7），信息冗余 |
| `ret_5d_std` | 与 `rvol_20` 度量同一维度（短期波动），冗余 |
| `high_low_20` | 与 `rvol_20` 度量同一维度（价格波动范围），冗余 |
| `pe_ttm` | 与 `inv_pb` 同为价值因子，信息重叠 |

#### 新增特征（5 个）

| 新特征 | 大类 | 计算方式 | 预期作用 |
|--------|------|---------|----------|
| `ind_rel_mom` | 动量 | stock_mom - industry_avg_mom | 剥离行业效应，提取个股 alpha |
| `mom_sharpe` | 动量 | mom_12_1 / rvol_20 | 波动率调整后动量，信噪比更高 |
| `market_beta` | 风险 | 60 日 stock vs market 回归斜率 | 捕捉系统性风险暴露 |
| `mkt_trend` | 市场状态 | 市场 20dMA / 60dMA | 趋势判断，牛熊市区分 |
| `mkt_vol` | 市场状态 | 市场 20 日波动率 | 波动率状态，高低波切换 |

| 参数 | 值 |
|------|----|

| 选股比例 | 前 5% |
| 行业约束 | 每行业最多 5 只 |
| 持仓缓冲 | +0.3σ |
| 市值范围 | 前 85% |
| LightGBM | num_leaves=31, lr=0.05, depth=4 |

### 优化预期

- 去冗余后模型更不容易过拟合，泛化能力提升
- 行业相对动量剥离行业 beta，提取纯个股 alpha
- 波动率调整动量提高信噪比，减少高波动噪声干扰
- 市场 beta 帮助模型感知系统性风险暴露
- 市场状态因子让模型区分牛熊市，自适应调整选股偏好
