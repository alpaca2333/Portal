# 行业中性集中选股策略 v2 回测报告（双周调仓 + ROE风控 + 缓冲带）

**策略文件**：`strategies/factor/industry_neutral_concentrated_v2.py`  
**数据来源**：`processed/stocks.db` (SQLite)  
**回测区间**：2022-01-01 ~ 2026-02-28  
**调仓频率**：双周（每月 15 日 + 月末）  
**基准**：上证指数 (sh000001) / 中证500 (sz000905)  
**股票池**：SH+SZ，排除 BJ，自由流通市值前 70%  
**行业中性化**：申万一级行业，因子在行业内 z-score  
**选股方式**：综合得分前 5%，每行业最多 5 只，等权持有  
**平均持仓**：106 只 / 28 个行业  

## 策略改进（相对原版）

### 改进 1：ROE 风控过滤

- **规则**：排除 ROE_TTM < -20.0% 的股票
- **定位**：纯风控过滤，不参与因子打分
- **目的**：剔除极端亏损公司（资不抵债、财务造假、ST预警）
- **设计理念**：
  - 经验证，将 ROE 作为独立打分因子会与 1/PB 产生对冲效应，降低收益
  - 作为过滤器仅移除极端差的公司，不影响正常选股逻辑
  - ROE 数据缺失的股票不被排除（不惩罚数据缺失）

### 改进 2：换仓缓冲带

- **规则**：上期持仓股票在打分时获得 +0.3σ 的 score bonus
- **目的**：减少边际换手——那些得分接近门槛的持仓股不会因微小波动被踢出
- **预期效果**：
  - 降低换手率 → 降低交易成本
  - 提高持仓连续性 → 更适合实盘操作
  - 不影响真正的 alpha 轮换（得分大幅下降的股票仍会被替换）

## 因子设计（与原版完全相同）

| 因子 | 表达式 | 权重 | 方向 |
|------|--------|------|------|
| 12-1 动量 | `close_t-20 / close_t-250 - 1` | 25% | 正向 |
| 估值 (1/PB) | `1 / PB` | 25% | 正向 |
| 量价确认 | `vol_ma20 / vol_ma120` | 15% | 正向 |
| 低波动 | `std(daily_ret, 20)` | 15% | 负向 |
| 小市值 | `log(free_market_cap)` | 10% | 负向 |
| 短期反转 | `close_t-10 / close_t - 1` | 10% | 正向 |

## 改进效果分析

### ROE 风控过滤效果

- 排除门槛：ROE < -20.0%
- 被排除的主要是：ST公司、连续亏损公司、资不抵债公司
- 预期效果：减少极端亏损标的，降低尾部风险

### 缓冲带效果

- Buffer 大小：+0.3σ（约相当于综合得分增加 0.3 个标准差）
- 预期效果：边际股票（得分接近入选/淘汰线的）更倾向于留在组合中
- 真正变差的股票（得分大幅下降）仍然会被换出

## 后续改进方向

- 因子 IC 动态加权：根据近期因子表现调整权重
- ROE 变化率因子：用 ΔROE 环比改善替代/补充静态 ROE
- 调仓缓冲带参数敏感性测试：尝试 0.2σ / 0.4σ / 0.5σ
- 止损机制：个股跌幅超阈值强制卖出
