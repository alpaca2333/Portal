# lgbm_ensemble_adaptive_v2 回测报告

## 1. 策略概述

### 策略思路

自适应集成选股 V2：波动率目标制 + 多级择时 + 自适应止损。基于 V1 改进，核心变化在仓位管理和风控层，选股层沿用 V1。

### 核心改进

1. **波动率目标制**：目标年化波动率 18%，连续平滑调仓，替代 V1 离散五档仓位
2. **三级趋势确认**：短期(MA5/MA20) + 中期(MA20/MA60) + 长期(新高新低比率)，2级以上确认才判定牛/熊
3. **自适应止损**：阈值 = f(个股波动率)，范围 [-20%, -8%]
4. **风格自适应**：动态 small_cap_bonus，根据大/小盘 60 日累计收益差切换
5. **训练优化**：4年窗口 + 指数衰减样本加权(半衰期12月) + 每6期重训练
6. **组合优化**：动态 softmax 温度(牛6.0/中4.0/熊2.0)，行业上限4只，最少4个行业
