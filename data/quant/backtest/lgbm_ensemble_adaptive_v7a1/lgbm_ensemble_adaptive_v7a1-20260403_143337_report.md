# lgbm_ensemble_adaptive_v7a1 回测报告

## 1. 策略概述

### 策略思路

自适应集成选股 V7a1（风格择时版）：在V7a（仓位管理修复版）基础上新增**风格择时P0**——根据CSI500/CSI300相对强弱动态调整小盘加分。

### V7a1 核心改进：风格择时

**自适应小盘加分 (Style Timing)**：
- 计算CSI500/CSI300的20日收盘价比值
- 与其60日均值比较，得到风格系数 style_coeff ∈ [-1, +1]
- 小盘强势(+1)：bonus = 0.02（满额）
- 风格均衡(0)：bonus = 0.010（半额）
- 大盘强势(-1)：bonus = 0（取消）
- 公式：effective_bonus = 0.02 × max(0, (1 + style_coeff) / 2)

### V7a 继承项：仓位管理修复

- **Cash-Aware真正生效**：通过注入虚拟CASH持仓绕过broker归一化
- 弱熊coeff=0.3时30%投资+70%现金，恐慌coeff=0.1时10%投资+90%现金

### V7 继承项：风控层

- 放宽恐慌触发（广度<15%即触发）
- 组合级回撤熔断（>15%回撤 → 缩减至50%）
- 最小持仓期（2期/4周）
- 增强换手缓冲（sigma=1.0）
- 收紧止损（-12%）

### V5 继承项（因子层+模型层）

- 3个行为Alpha因子：ivol、max_ret_20d、skew_20d
- 连续化行业动量（scale=0.02）
- 行业中性标签 + 双模型融合 SM(0.6)+CS(0.4)
- 市场状态择时 + softmax加权 + 个股止损
