# lgbm_ensemble_adaptive_v7a 回测报告

## 1. 策略概述

### 策略思路

自适应集成选股 V7a（仓位管理修复版）：修复V7中现金仓位管理未生效的严重Bug。V7的Cash-Aware Sizing代码虽然将权重缩放到sum=final_coeff，但回测引擎broker.rebalance()会将权重重新归一化回sum=1.0，导致现金比例完全失效。V7a通过注入虚拟CASH持仓占位的方式绕过引擎归一化，真正实现现金仓位管理。

### V7a 修复项

- **Cash-Aware真正生效**：弱熊coeff=0.3时，策略只将30%资金分配给股票，70%保持现金。V7中这个功能完全无效（被引擎归一化抵消）
- **修复方式**：在target_weights中注入虚拟CASH条目（权重=1-final_coeff），broker因找不到CASH的价格而跳过买入，资金自然留为现金

1. **现金仓位管理 (Cash-Aware Sizing)**：仓位系数不仅控制持仓数量，还控制实际资金分配比例。弱熊coeff=0.3意味着仅30%资金投入+70%现金，而非V5的仅减少持仓数但仍满仓运作
2. **放宽恐慌触发**：恐慌模式从'下跌+广度<30%+高波动>1.5'放宽为'下跌+广度<15%'，移除波动率要求。因为2024年2月崩盘时波动率尚未飙升，但广度已经崩溃，V5未能及时触发恐慌
3. **组合级回撤熔断**：组合净值从峰值回撤>15%时，仓位强制缩减至50%。恢复条件：回撤<5%。直接应对V5在2024H1无组合级风控覆盖的问题

#### P1: 换手率控制
4. **最小持仓期**：持仓不满2期（4周）的股票获得3倍换手缓冲加分，大幅降低短期换手。目标：将V5的41.5%单期持仓率显著降低
5. **增强换手缓冲**：buffer_sigma从0.5提高到1.0，加倍惩罚不必要的换仓
6. **收紧个股止损**：从V5的-20%收紧至12%，与组合熔断互补

### V5 继承项（因子层+模型层）

- 3个行为Alpha因子：ivol（特质波动率）、max_ret_20d（彩票因子）、skew_20d（偏度因子）
- 连续化行业动量（rank-normalized评分，scale=0.02）
- 行业中性标签（训练时减去行业中位数收益）
- 双模型融合：Smart Money (0.6) + Cross-Sectional (0.4)
- 共识过滤（综合前5% 且至少一个模型前10%）
- 市场状态择时：五档仓位(1.0/0.8/0.5/0.3/0.1) + 现金比例控制
- softmax 加权 (temperature=5.0)，单股上限8%
- 个股止损(-12%) + 冷却 + 换手缓冲
