# lgbm_multifactor 回测报告

## 1. 策略概述

### 策略思路

基于 LightGBM 梯度提升树的截面多因子 ML 选股策略。使用 15 个特征（6 基础 + 9 扩展），通过 3 年滚动窗口训练，预测截面排名分位数（而非绝对收益），消除收益分布非平稳性。在全市场沪深主板中选取自由流通市值前 70% 的股票，按 ML 预测分数取前 5%，每行业最多 5 只，等权持有，双周调仓。

### 因子列表（15个特征）

| 因子 | 描述 | 类别 |
|------|------|------|
| inv_pb | 市净率倒数 | 价值 |
| roe_ttm | ROE（TTM） | 质量 |
| pe_ttm | 市盈率（TTM） | 价值 |
| log_cap | 对数自由流通市值 | 规模 |
| mom_12_1 | 12月动量（跳过最近1月） | 动量 |
| turnover_20 | 20日换手率 | 流动性 |
| mom_3_1 | 3月动量（跳过最近1月） | 短期动量 |
| mom_6_1 | 6月动量（跳过最近1月） | 中期动量 |
| rev_10 | 10日反转 | 反转 |
| rvol_20 | 20日已实现波动率 | 风险 |
| ret_5d_std | 5日收益标准差 | 微观波动 |
| volume_chg | 量比变化（5日/20日） | 量价 |
| high_low_20 | 20日振幅比 | 风险 |
| close_to_high_60 | 收盘价距60日高点 | 价格距离 |
| vol_confirm | 动量×量变交互项 | 量价确认 |

### 技术细节

1. **特征预处理**：所有特征在每期截面内做排序归一化映射到 [0,1]
2. **标签构建**：截面排序标签（forward return 百分位排名）
3. **训练窗口**：3 年滚动窗口，每 4 个调仓周期重训一次
4. **Purge**：训练数据 forward return 计算不超出训练窗口末尾
5. **模型**：LightGBM 回归模式，预测排名分位
6. **选股**：ML 分数前 5%，每行业最多 5 只
7. **缓冲带**：持仓股 ML 分数 +0.3σ 加分，降低换手
8. **选股范围**：沪深主板，自由流通市值前 70%
9. **数据访问**：全部通过 DataAccessor，受前视偏差保护

### 已知局限

- 训练窗口固定 3 年，未做窗口长度敏感性测试
- 未使用交叉验证选超参
- 未做因子 IC/IR 分析
- 未考虑涨跌停限制
