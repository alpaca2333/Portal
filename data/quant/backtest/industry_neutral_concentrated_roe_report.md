## 策略逻辑

本策略将原版的独立 VALUE(1/PB) 因子替换为 **PB-ROE 复合因子**：

### 核心改进：PB-ROE 复合因子

- **公式**：`ROE_TTM / PB`（ROE per unit of PB）
- **目的**：直接选出"高性价比"公司——盈利能力强但估值不贵
- **优势**：避免独立 ROE + 1/PB 的对冲效应（高ROE公司通常PB也高）
- **过滤**：ROE<=0 或 PB<=0 的公司不参与打分（排除亏损和破净异常）
- **权重**：30%（获得原 VALUE 25% 的预算 + 额外 5%）

## PB-ROE 复合因子的作用机制

1. **消除对冲效应**：独立的 1/PB 和 ROE 因子会互相抵消（高ROE→高PB→低1/PB），复合因子直接衡量性价比
2. **过滤低估值陷阱**：低PB但ROE也低的公司（破产边缘/周期底部），ROE/PB 不会很高
3. **过滤泡沫股**：高ROE但PB极高的公司（如100x PB的概念股），ROE/PB 也不会突出
4. **选出真正的价值股**：ROE=20% + PB=2x → ROE/PB=10，远优于 ROE=5% + PB=0.5x → ROE/PB=10

## 下一步改进方向

- 换仓缓冲带（Buffer）：进入 top 5%，退出 top 8%，减少边界股票频繁进出
- 因子 IC 监控：每期计算各因子 Rank IC，追踪因子有效性
- 得分加权：高分股票给更大权重，进一步集中 alpha
- ROE 变化率因子：用 ΔROE 环比改善替代/补充静态 ROE
