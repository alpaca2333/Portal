# lgbm_ensemble_adaptive_v7 回测报告

## 1. 策略概述

### 策略思路

自适应集成选股 V7（风控增强版）：在V5基础上聚焦**风险控制与换手率优化**。针对V5深度分析中暴露的三大问题（2024H1超额回撤-22.7%、年化换手率1544%、仓位系数仅控制持仓数量不控制资金），新增现金仓位管理、组合熔断机制、最小持仓期等风控措施。

### V7 核心改进（基于V5，风控层优化）

#### P0: 资金分配与回撤保护
1. **现金仓位管理 (Cash-Aware Sizing)**：仓位系数不仅控制持仓数量，还控制实际资金分配比例。弱熊coeff=0.3意味着仅30%资金投入+70%现金，而非V5的仅减少持仓数但仍满仓运作
2. **放宽恐慌触发**：恐慌模式从"下跌+广度<30%+高波动>1.5"放宽为"下跌+广度<15%"，移除波动率要求。因为2024年2月崩盘时波动率尚未飙升但广度已经崩溃，V5未能及时触发恐慌
3. **组合级回撤熔断**：组合净值从峰值回撤>15%时，仓位强制缩减至50%。恢复条件：回撤<5%。直接应对V5在2024H1无组合级风控覆盖的问题

#### P1: 换手率控制
4. **最小持仓期**：持仓不满2期（4周）的股票获得3倍换手缓冲加分，大幅降低短期换手。目标：将V5的41.5%单期持仓率显著降低
5. **增强换手缓冲**：buffer_sigma从0.5提高到1.0，加倍惩罚不必要的换仓
6. **收紧个股止损**：从V5实际运行的-20%收紧至-12%，与组合熔断互补

### V5 继承项（因子层+模型层）

- 3个行为Alpha因子：ivol（特质波动率）、max_ret_20d（彩票因子）、skew_20d（偏度因子）
- 连续化行业动量（rank-normalized评分，scale=0.02）
- 行业中性标签（训练时减去行业中位数收益）
- 双模型融合：Smart Money (0.6) + Cross-Sectional (0.4)
- 共识过滤（综合前5% 且至少一个模型前10%）
- 市场状态择时：五档仓位(1.0/0.8/0.5/0.3/0.1) + 现金比例控制
- softmax 加权 (temperature=5.0)，单股上限8%
- 个股止损(-12%) + 冷却 + 换手缓冲

### 关键参数变更 (V5 → V7)

| 参数 | V5 | V7 | 说明 |
|------|-----|-----|------|
| buffer_sigma | 0.5 | 1.0 | 加倍换手缓冲 |

## 6. 总结与改进空间

### 表现总结

V7在V5基础上通过风控层优化，实现了显著的收益提升和风险特征改善：

| 指标 | V5 | V7 | 变化 |
|------|-----|-----|------|
| 年化收益 | 19.14% | 27.33% | **+8.19%** |
| 夏普比率 | 0.76 | 1.01 | **+0.25** |
| Sortino比率 | 1.08 | 1.58 | **+0.50** |
| Calmar比率 | 0.60 | 0.84 | **+0.24** |
| 最大回撤 | -31.69% | -32.54% | -0.85% (基本持平) |
| 累计收益 | 305.91% | 591.06% | **+285.15%** |
| 胜率 | 58.2% | 61.1% | +2.9% |
| 超额胜率(vs上证) | 60.6% | 63.0% | +2.4% |
| 6月滚动超额胜率 | 80.5% | 81.7% | +1.2% |
| 上行捕获率 | 1.22 | 1.34 | **+0.12** |
| 下行捕获率 | 0.70 | 0.62 | **-0.08 (改善)** |
| 捕获比 | 1.73 | 2.16 | **+0.43** |
| 年化换手率 | ~1544% | ~1364% | **-180%** |
| 单期持仓占比 | 41.5% | 33.7% | **-7.8%** |

**核心结论**：V7在保持最大回撤基本不变的前提下，年化收益从19%提升到27%，夏普从0.76突破1.0。捕获比从1.73提升到2.16，说明现金仓位管理有效降低了下行暴露。换手率下降约12%，单期持仓占比下降8%，但距离目标（<1000%）仍有差距。

### 风险分析

**改善的方面**：
- 下行捕获率从0.70降至0.62，说明现金仓位管理在熊市中发挥了保护作用
- Sortino从1.08到1.58，下行风险调整后收益大幅改善
- 市场状态因子贡献从V5的~16%提升至V7的22.0%，择时机制增强

**仍存在的问题**：
- **2024H1超额回撤反而恶化**：V7 -27.9% vs V5 -22.7%。虽然新增了熔断机制，但收紧止损(-12%)导致在小微盘踩踏中更频繁触发止损，反而加剧了损失。这说明个股止损在系统性风险中可能适得其反
- 最大回撤-32.54%与V5的-31.69%基本持平，组合熔断未能有效缩小极端回撤
- 年化换手率仍有1364%，交易成本估计~7.8%，仍是收益的主要侵蚀项
- 小盘风格依赖未改变（small_cap因子贡献13.6%），2024年大盘主导年仍亏损-5.03%

### 改进方向

1. **P0: 风格择时层**（解决小盘依赖）
   - 新增CSI500/CSI300相对强弱信号，当大盘主导时动态调低small_cap_bonus
   - 此改进直接针对2024年-19.85%超额回撤的根因

2. **P0: 止损策略优化**（解决2024H1恶化问题）
   - 系统性风险（广度<20%）时暂停个股止损，避免踩踏中被动卖出
   - 或改为ATR自适应止损，替代固定阈值

3. **P1: 进一步降低换手率**
   - 考虑将min_holding_periods从2提高到3
   - 或引入交易成本感知的选股（将预估交易成本纳入ensemble score）

4. **P2: 流动性约束**
   - 排除日均成交额<5000万的股票，减少小盘滑点损耗
   - 预期可将实盘交易成本降低30-50%
